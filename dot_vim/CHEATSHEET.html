Vim Cheat Sheet

Leader key = ,  ·  Local leader = \  ·  e.g. <leader>w → ,w

Motion

w / bNext / prev word start
e / geNext / prev word end
W / B / ESame but WORD (whitespace-delimited)
0 / ^ / $Line col-0 / first non-blank / end
gg / GTop / bottom of file
50G  or  :50Jump to line 50
%Jump to matching bracket / #ifdef
{ / }Prev / next empty line (paragraph)
Ctrl-D / Ctrl-UHalf-page down / up
H / M / LTop / middle / bottom of screen
zz / zt / zbScroll cursor to middle / top / bottom
f{c} / F{c}Jump to next / prev {c} on line
t{c} / T{c}Jump before next / prev {c}
; / ,Repeat / reverse last f/t jump
* / #Search fwd / bwd for word under cursor
'' (two ticks)Jump back before last big jump
Ctrl-O / Ctrl-IWalk backward / fwd through jump list

Text Objects — the vim superpower

Pattern: {op}{scope}{object}  e.g. ci" = change inside quotes

Objects

iw / awInner word / a word (incl. space)
i" / a"Inside / around double quotes
i' / a'Inside / around single quotes
i( / a(Inside / around parentheses
i{ / a{Inside / around braces
i[ / a[Inside / around brackets
it / atInside / around HTML/XML tag
ip / apInner / a paragraph

Common combos

ciwChange word under cursor
ci"Change text inside quotes
ca(Change including the parentheses
di{Delete inside braces
yi[Yank inside brackets
=i{Re-indent inside braces (e.g. a function)
gUiw / guiwUppercase / lowercase word

Operators

All compose with text objects and motions: d3j, y$, c%, >ip

d / c / yDelete / change / yank
> / <Indent / dedent
=Auto-indent
gU / guUppercase / lowercase
gcToggle comment (NERDCommenter)

Insert Mode

i / aInsert before / after cursor
I / AInsert at line start / end
o / ONew line below / above, enter insert
s / SDelete char / line and insert
CChange to end of line
Ctrl-WDelete word backward
Ctrl-UDelete to start of line
Ctrl-R {reg}Paste register without leaving insert
Ctrl-R =Insert expression result (e.g. 2+2→4)

Visual Mode

v / VCharacter / line visual
Ctrl-VBlock / column visual
oMove to other end of selection
gvRe-select last visual selection
I (block)Insert at start of every selected line
A (block)Append at end of every selected line

Registers

Prefix any yank or paste with "{reg} to target a register

"ayy / "apYank line into / paste from register a
"+y / "+pYank to / paste from system clipboard
"0pPaste last yanked text (ignores deletes)
"_dDelete into black hole (keeps clipboard clean)
:regShow all registers
Ctrl-R {reg}Paste register in insert / command mode

Special: "0 last yank · "+ clipboard · "% filename · ": last command · "/ last search

Marks

m{a}Set mark a at cursor
`{a}Jump to exact position of mark a
'{a}Jump to line of mark a
`. (backtick .)Jump to position of last change
`[ / `]Start / end of last yank or change
`< / `>Start / end of last visual selection
:marksList all marks

Uppercase marks (mA–mZ) are global — work across files

Macros

q{a}Start recording macro into register a
qStop recording
@{a}Play macro a
@@Replay last macro
10@aPlay macro a 10 times
:reg aInspect contents of macro a

Tip: end the macro by moving to the next target line, then 10@@ to sweep across many lines

Search & Replace

/patternSearch forward
?patternSearch backward
n / NNext / prev match
:%s/old/new/gReplace all in file
:%s/old/new/gcReplace all, confirm each
:'<,'>s/old/new/gReplace in visual selection

\v prefix enables "very magic" — most chars special (closer to PCRE). e.g. :%s/\v(\w+)/[\1]/g

Command Line

:e fileOpen file
:w !sudo tee %Save file opened without sudo
:r !cmdInsert shell command output below cursor
:%!cmdFilter entire file through shell command
:g/pat/dDelete all lines matching pattern
:v/pat/dDelete all lines not matching
:g/pat/norm @aRun macro a on every matching line
Ctrl-R Ctrl-WInsert word under cursor into command
q: or Ctrl-FOpen editable command history window
:earlier 5mUndo to file state 5 minutes ago
:later 5mRedo to state 5 minutes forward
